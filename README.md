# speedloggerz
Data keepsafe for all things SpeedRunners

# Prerequisites: 
- Angular
- npm
- GoLang
- NodeJS 

# After cloning
- run `npm install --force --legacy-peer-deps` in the root directory.
- run `ng serve` in the root directory.

# Then
- run the Go Server, 
- cd into: `cd server`
- run `go run main.go`
- load browser and go to localhost:4200
